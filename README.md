# Frontend
## Install dependencies

```sh
npm install
```

### Compile and Hot-Reload

```sh
npm run dev
```

# Backend

## Install dependencies

```sh
cd backend
npm install
```

## Run node.js server

```sh
cd backend
node app.js
```
